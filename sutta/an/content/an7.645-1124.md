Numbered Discourses 7

11. Abbreviated Texts Beginning With Greed

Untitled Discourses on Hate, Etc.

“Of hate …

delusion …

anger …

acrimony …

disdain …

contempt …

jealousy …

stinginess …

deceitfulness …

deviousness …

obstinacy …

aggression …

conceit …

arrogance …

vanity …

for insight into negligence …

complete understanding …

complete ending …

giving up …

ending …

vanishing …

fading away …

cessation …

giving away …

letting go of negligence

these seven things should be developed.”

That is what the Buddha said.

Satisfied, the mendicants approved what the Buddha said.

The Book of the Sevens is finished.
