
这里搜集了部分在网络上能够找到的佛教早期经典，包括南传巴利经藏[Nikaya]的中文译本以及北传经典[阿含经】。

期望读者通过阅读佛教早期经典，能够了解到佛陀时期修行者的修行目的和修行方法，从而为自己的人生重新明确方向。

本站只做简单的搜集整理，未经严格校对，文字可能存在错漏。经文所有权归属于三宝，译文版权归译者或出版方所有，如有侵权请留言，收到后即删除。
