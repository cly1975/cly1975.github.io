# 巴利经藏 | Nikaya
这里搜集了部分在网络上能够找到的国内优秀的巴利经藏的中文译本。这些译本都是由专业学人直接译自巴利语、并且至少完整译出了5部中的一部。
其中有些通过网络发布，有些已经通过出版社出版发行。本站只做简单的搜集整理，未经严格校对，文字可能存在错漏，提供给有志于佛陀古道的学人参考。
经文所有权归属于三宝，译文版权归译者或出版方所有，如有侵权请留言，收到后即删除。
